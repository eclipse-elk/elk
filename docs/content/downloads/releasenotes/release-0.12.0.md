## Details

This release concentrates on bug fixes and enhancements concerning many details of the layout algorithms as well as some cleanup and upgrades under the hood. See GitHub for the full [list of resolved issues](https://github.com/eclipse-elk/elk/milestone/22?closed=1).

### New Features and Enhancements
- [#1194](https://github.com/eclipse-elk/elk/pull/1194): Add option to run a second node placer after using the Network node flexibility option.

### Changes
- [#1170](https://github.com/eclipse-elk/elk/pull/1170): Update condition for layer splitting logic to make it more flexible.

### Licensing
- [#1185](https://github.com/eclipse-elk/elk/issues/1185), [#1191](https://github.com/eclipse-elk/elk/pull/1191), [#1184](https://github.com/eclipse-elk/elk/pull/1184): Add GPL-3.0 as a secondary license.
